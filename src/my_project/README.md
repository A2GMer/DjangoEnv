# 🎉 シンプルで楽しいDjangoアプリへようこそ！🎉

このアプリは、ユーザーごとに140字以内の文字列を登録、削除、編集できる、超シンプルなDjangoアプリです！  
Twitterのつぶやきみたいな感覚で、自由にメッセージを楽しめちゃうよ！
